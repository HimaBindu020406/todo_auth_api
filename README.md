# To-Do List REST API with Authentication

This project implements a To-Do List REST API using Node.js and Express.  
MongoDB Atlas is used as the database, and JWT is used for authentication.

## Features
- User Registration
- User Login
- JWT-based Authentication
- Create and View To-Do Tasks

## Technologies Used
- Node.js
- Express
- MongoDB Atlas
- Mongoose
- JWT
- bcryptjs
- Postman

## How to Run
1. Clone the repository
2. Install dependencies using `npm install`
3. Add a `.env` file with MongoDB URI
4. Run `npm start`
